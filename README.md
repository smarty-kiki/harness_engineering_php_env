# harness_engineering_php_env
PHP 项目的 Harness Engineering 开发环境

## debian_php_dev_env

基于 Debian trixie 的 PHP 开发环境容器

- nginx
- mariadb-server
- redis-server
- mongodb
- beanstalkd
- php8.4-fpm
- phpunit
- inotify-tools
- supervisor
- wget
- git
- composer
- vim
- tmux
- pip
- mycli
- claude (Anthropic CLI)
